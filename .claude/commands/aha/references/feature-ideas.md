# Example Ideas by Feature Type

> Reference document for /aha command. Common aha moment patterns by feature type.

## Data Tables

* Inline actions on row hover
* Saved filter presets per role
* Sticky headers with column resize
* Keyboard navigation (up/down/Enter)
* Export only selected rows

## Forms

* Auto-save drafts
* Duplicate detection before submit
* Smart validation with real examples
* Template / preset buttons

## Dashboards

* Click-to-drill-down charts
* Comparison mode (vs yesterday / last week)
* Anomaly highlights
* Quick action buttons per widget

## Lists & Search

* Recent searches
* Suggested filters based on data
* Bulk actions with preview
* Infinite scroll with position memory

## User Management

* Role preview before assignment
* Activity timeline
* Quick impersonate (admin only)
* Password strength meter with suggestions

## Settings

* Change preview before save
* Reset to defaults per section
* Import/export configuration
* Audit log for changes

## API / Microservice

* Request/Response logging toggle
* Rate limit status indicator
* API key rotation with grace period
* Health check dashboard
